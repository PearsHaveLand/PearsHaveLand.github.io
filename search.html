This page was inspired by account creation forms throughout the Internet. On many sites, you must select a country of residence via a dropdown menu. I find long dropdowns frustrating and inconvenient. I know some sites use a search bar solution, so I decided to implement my own version.

So far, there is no click functionality. All that's working at the moment is just text search and highlight.
